## 4. Design System

### Couleurs (deja configurees dans global.css)

| Variable | Light Mode | Dark Mode | Usage |
|----------|------------|-----------|-------|
| `--primary` | #058e9f (Teal) | Teal lumineux | Boutons, liens, accents |
| `--secondary` | Gris clair | Gris fonce | Arriere-plans secondaires |
| `--background` | Blanc | Noir profond | Fond de page |
| `--foreground` | Noir | Blanc | Texte principal |
| `--accent` | Teal clair | Teal sombre | Hover, focus |
| `--muted` | Gris tres clair | Gris fonce | Texte secondaire |
| `--destructive` | Rouge | Rouge clair | Erreurs, alertes |

### Palette HEX de Reference
- **Primary Teal**: #058e9f
- **Dark/Black**: #020202
- **White**: #ffffff
- **Light Gray**: #f5f5f5

### Typographie
- Police principale: Geist Sans (deja configuree)
- Police mono: Geist Mono
- Hierarchie: Titres larges et bold, corps de texte lisible

### Bordures
- Radius de base: 0.625rem (10px)
- Style Apple: coins arrondis genereux sur les cartes

### Animations (Motion)
- **Principes Apple**:
  - Subtiles et naturelles
  - 60fps fluide
  - Ease-out pour les entrees
  - Ease-in pour les sorties
  - Duree: 200-400ms pour la plupart
  - Micro-interactions sur hover/focus
